Review code with findings first. Prioritize bugs, regressions, missing tests, unsafe assumptions, and convention violations.

Use the plugin standards as review lenses when relevant:

- coding style
- function naming
- DTO/domain boundary
- TanStack Query patterns
- test quality

Do not rewrite code unless explicitly asked.
